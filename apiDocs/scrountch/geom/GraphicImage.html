Package: scrountch.geom

[Groovy] Class GraphicImage

      • scrountch.geom.GraphicImage

      • A graphic Object that handles an Image. The iconImage data is loaded through an IconImage object. Other data specific to instances of this class:

        • the initial coordinates position (coorX, coorY) though they can be changed this is discouraged
        • the scale scale that can be changed any time
        The same graphic Image can be shared across different Cells of the same size but not if Cells have different sizes
