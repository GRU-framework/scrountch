[Groovy] Class Cell

        • scrountch.geom.Cell

        • a Canvas with special features. It contains a list of GraphicObject that are drawn each time an instance of this class is repaint.
          Implementation note: there is a circular dependency of this class with GraphicObject and its subclasses. Beware when modifying.
