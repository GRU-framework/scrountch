Package: scrountch.geom

[Groovy] Trait I18NMethodNaming

    • scrountch.geom.I18NMethodNaming

    • This trait should be implemented by all classes that support internationalisation of method names.
      The internationalisation should be configured that way:

      • the base name for ResourceBundle is "scrountch/methods" (so for instance the property file for french is "scrountch/methods_fr.properties"
      • the entries in this resources are of the form SimpleOriginalClassName.translatedMethod = realMethodName. So for instance in french we have: 
                  # SFrame
          SFrame.canevas=getCell
          SFrame.tousLesCanevas=cells
          SFrame.ajouterAuNord=addNorth
