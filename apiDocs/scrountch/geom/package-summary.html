Package scrountch.geom

Package scrountch.geom Description

This package deals with programming tools for an easier access to java.awt features. The main goal is to provide a learning environment for would-be programmers. All codes are Groovy codes: simple programming paradigms are preferred to performance enhancements. for a more advanced description see the scrountch programming tutorial
